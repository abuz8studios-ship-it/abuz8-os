7. Core Schemas

PostgreSQL Memory Table

CREATE TABLE memories (
  id UUID PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  memory_type TEXT,
  brain_region TEXT,
  modality TEXT,
  timestamp TIMESTAMPTZ DEFAULT now(),
  salience FLOAT DEFAULT 0.5,
  confidence FLOAT DEFAULT 0.5,
  utility FLOAT DEFAULT 0.5,
  recurrence FLOAT DEFAULT 0.0,
  source TEXT,
  embedding_id TEXT,
  metadata JSONB
);

10. Exact Build Steps

Phase 1 — Skeleton

React frontend, FastAPI backend, Docker Compose for Postgres/Neo4j/Qdrant/Redis.

Phase 2 — Memory API

Create endpoints: write memory, search memory, link memory, score salience, retrieve context.

Phase 3 — Brain UI

Build clickable brain map with modules: executive, memory, harmonic, visual, language, salience, motor, reflection.

Phase 4 — Harmonic Engine

Add audio upload, extract BPM/key/chords/pitch/chroma, store as memories and graph nodes.

Phase 5 — Agent Router

Route tasks to Claude/GPT/Gemini/local models based on task type, cost, confidence, and privacy.

Phase 6 — Reflection

After every task, create: result, failure, cause, fix, reusable rule, memory links.

Phase 7 — Consolidation

Nightly job compresses memories, merges duplicates, strengthens graph edges, decays stale data.

Phase 8 — Proof

Run benchmark tasks: recall, music analysis, graph reasoning, tool execution, self-correction.
